# 🤖 Robotics & AI Portfolio
**A Comprehensive Collection of Intelligent Systems, Machine Learning, and Robotics Projects.**

Welcome! This repository serves as a consolidated portfolio of my work as a Robotics and Intelligent Systems student. It bridges the gap between hardware integration, supervised/unsupervised machine learning, and advanced computer vision.

---

## 📁 Repository Structure

### ⚙️ [C++ and Hardware](./C++_and_Hardware/)
*Focuses on microcontroller development, sensor fusion, and real-time automation.*
*   **Smart Parking System:** An automated gate entry system utilizing Arduino Uno, IR sensors, and C++. Features real-time slot tracking and LCD status updates.
*   **Environmental Monitoring:** Integrated DHT and Ultrasonic sensors for obstacle detection and climate data acquisition.

### 📊 [Machine Learning](./Machine_Learning/)
*Data-driven insights and predictive modeling using Scikit-Learn and Python.*
*   **Heart Disease Prediction:** A classification system trained on the UCI clinical dataset to assist in medical diagnostics.
*   **Anomaly Detection (DBSCAN):** Identifying mechanical failures in robotic sensor datasets through statistical feature engineering and unsupervised clustering.
*   **Supervised Learning Suite:** Projects exploring K-Means, PCA, and Gaussian Mixture Models for data dimensionality reduction.

### 🧠 [Deep Learning](./Deep_Learning/)
*Neural networks and Computer Vision applications.*
*   **Robotic Grasping CNN:** A custom Convolutional Neural Network designed to predict optimal grasp coordinates from images, featuring a Gradio web interface for real-time testing.
