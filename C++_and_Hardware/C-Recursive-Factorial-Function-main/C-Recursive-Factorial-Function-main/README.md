# C++ Recursive Factorial Calculator

A clean and beginner-friendly C++ program that calculates the factorial of a number using recursion.  
This project demonstrates function recursion, base case handling, and input validation.

---

## Features
- Uses a recursive function for factorial calculation  
- Handles base cases (`0!` and `1!`) correctly  
- Validates negative input values  
- Outputs results in a clear, readable format  

---

## Concepts Used
- Recursion  
- Base Cases & Recursive Calls  
- Basic Input/Output  
- Error handling for invalid input  

---

## How to Run
1. Save the file as `main.cpp`  
2. Compile the program using:  
   ```bash
   g++ main.cpp -o recursive_factorial
