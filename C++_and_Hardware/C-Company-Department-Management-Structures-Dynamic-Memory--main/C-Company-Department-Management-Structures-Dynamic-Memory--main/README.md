# C++ Company & Department Management System

A simple C++ program showcasing the use of nested structures and dynamic memory allocation to store and manage company and department information.  
This project demonstrates structured data handling, pointer usage, and clean input/output formatting.

---

## Features
- Uses nested `struct` types (`Company` containing `Department`)  
- Dynamically allocates memory for company data  
- Accepts full company details including department and head name  
- Displays organized output for all entered information  

---

## Concepts Used
- `struct` composition (nested structures)  
- Dynamic memory allocation using `new` and `delete`  
- Pointer-based function parameters  
- Standard Input/Output with `getline`  

---

1. Save the program as `main.cpp`  
2. Compile the file:  
   ```bash
