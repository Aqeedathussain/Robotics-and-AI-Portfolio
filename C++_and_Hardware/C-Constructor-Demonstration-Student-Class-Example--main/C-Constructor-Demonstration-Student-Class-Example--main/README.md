# C++ Student Class Constructor Example

A minimal C++ program that demonstrates how constructors work in object-oriented programming.  
The project focuses on initializing class attributes through parameterized constructors and displaying object data.

---

## Features
- Uses a parameterized constructor for initializing class members  
- Demonstrates object creation and method invocation  
- Clean and simple structure ideal for beginners learning C++ OOP  

---

## Concepts Used
- Classes & Objects  
- Parameterized Constructors  
- Member Function (`display()`)  
- Basic Input/Output (`cout`)  

---

## How to Run
1. Save the program as `main.cpp`  
2. Compile the code using:  
   ```bash
   g++ main.cpp -o student_app
