# C++ Vowel or Consonant Checker

A beginner-friendly console-based C++ program that determines whether a given alphabet is a vowel or a consonant.  
This project highlights character handling, lowercase conversion, and validation using standard C++ libraries.

---

## Features
- Validates if the input is a valid alphabet character  
- Converts input to lowercase for consistent checking  
- Identifies vowels (`a, e, i, o, u`)  
- Categorizes all other alphabets as consonants  

---

## Concepts Used
- Input/Output (`cin`, `cout`)  
- Character validation using `<cctype>` (`isalpha`, `tolower`)  
- Conditional logic (`if-else`, logical OR checks)  

---

## How to Run
1. Save the program as `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o vowel_checker
