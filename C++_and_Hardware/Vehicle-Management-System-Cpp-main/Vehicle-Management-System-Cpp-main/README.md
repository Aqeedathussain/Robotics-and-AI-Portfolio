# Vehicle Management System (C++)

A console-based Vehicle Management System implemented in C++ using **Object-Oriented Programming**, **Inheritance**, **Friend Classes**, and **Encapsulation**.  
This system manages vehicle details, owner information, maintenance logs, traffic violations, insurance records, and performance diagnostics with admin-controlled editing functionality.

---

## Features

### Vehicle & Owner Management
- Stores vehicle registration number, model, and type
- Stores owner name, contact, email, and address

### Maintenance Log System
- Maintains multiple maintenance records
- Stores service type, date, and cost
- Displays detailed maintenance history

### Traffic Violation Records
- Stores multiple violation entries
- Records violation type, date, and fine

### Insurance Details
- Stores insurance company name
- Policy number & coverage status

### Performance Monitoring
- Tracks engine health
- Fuel efficiency
- System diagnostics

### Admin Panel
- Secure authentication using Admin ID
- Edit/update:
  - Vehicle Details  
  - Owner Details  
  - Maintenance Records  
  - Traffic Violations  
  - Insurance Information  
  - Performance Parameters  

---

---

## Program Flow
 1 User selects:
- Display all vehicle details  
**or**
- Edit information using Admin Access  

 2 If Admin Mode:
- Enter correct Admin ID  
- Choose which record to modify  
- Update fields through input  

3 Updated information is displayed
