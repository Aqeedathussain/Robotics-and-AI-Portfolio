# C++ Employee Salary Management

A beginner-friendly C++ program that demonstrates how to manage employee data using structures, references, and pointers.  
This project shows how to increase salary via pass-by-reference and display employee details through pointer access.

---

## Features
- Uses a `struct` to store employee details  
- Updates salary by 10% using pass-by-reference  
- Displays employee data using pointer-based function parameters  
- Clearly demonstrates references vs pointers in a real example  

---

## Concepts Used
- Structures (`struct`)  
- Pass-by-Reference (`Employee&`)  
- Pointer Access (`Employee*`)  
- Basic Output Formatting  

---
