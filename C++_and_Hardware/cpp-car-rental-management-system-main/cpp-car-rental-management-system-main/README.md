# Car Rental Management System (C++)

A beginner-friendly C++ project demonstrating Object-Oriented Programming with classes, dynamic associations, and vector-based storage.  
This console application allows adding cars, registering customers, checking availability, and renting cars interactively.

---

## Features
- Add multiple cars with brand, model, and availability status  
- Register customers and assign them to available cars  
- Track which customer rented which car  
- Display complete system records in an organized way  

---

## Concepts Used
- Classes & Objects  
- Encapsulation  
- Pointers & Dynamic Linking (Customer ↔ Car)  
- STL Vector  
- Condition-based availability handling
