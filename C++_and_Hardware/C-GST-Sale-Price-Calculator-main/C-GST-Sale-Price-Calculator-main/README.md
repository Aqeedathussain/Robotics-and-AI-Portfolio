# C++ GST Sale Price Calculator

A beginner-friendly C++ program that calculates the sale price of a product after adding GST.  
This project demonstrates arithmetic operations, constant usage, and user input/output handling.

---

## Features
- Accepts the cost price of a product from the user  
- Calculates the sale price including 15% GST  
- Displays the final sale price clearly  
- Simple and easy-to-understand console application  

---

## Concepts Used
- Arithmetic operations (`+`, `*`, `/`)  
- Constants (`const`) for GST rate  
- Input/Output handling (`cin`, `cout`)  
- Basic C++ program structure  

---

## How to Run
1. Save the file as `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o gst_calculator
