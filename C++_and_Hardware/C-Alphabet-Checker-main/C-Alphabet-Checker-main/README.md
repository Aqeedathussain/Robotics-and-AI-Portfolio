# C++ Alphabet Checker

A beginner-friendly C++ program to determine if a character entered by the user is an alphabet letter.  
This project introduces basic input handling and character validation using the standard C++ library.

---

## Features
- Accepts a single character as input  
- Validates if the input is an alphabet (A-Z or a-z)  
- Provides clear output indicating the result  

---

## Concepts Used
- Standard Input/Output (`cin`, `cout`)  
- Character Handling with `<cctype>` (`isalpha`)  
- Conditional Statements (`if-else`)  

---

## How to Run
1. Copy the code into a file named `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o alphabet_checker
