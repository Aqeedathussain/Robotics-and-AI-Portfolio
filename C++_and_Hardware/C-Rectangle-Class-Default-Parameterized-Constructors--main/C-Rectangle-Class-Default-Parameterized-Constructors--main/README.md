# C++ Rectangle Class Constructor Demonstration

A simple C++ program that demonstrates how default and parameterized constructors initialize class objects.  
The project calculates and displays the area of a rectangle using different constructor types.

---

## Features
- Implements a default constructor with preset values  
- Implements a parameterized constructor for custom dimensions  
- Calculates and displays rectangle area  
- Demonstrates constructor overloading  

---

## Concepts Used
- Classes & Objects  
- Default Constructor  
- Parameterized Constructor  
- Constructor Overloading  
- Basic arithmetic operations  

---

1. Save the program as `main.cpp`
