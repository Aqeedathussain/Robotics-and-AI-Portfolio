# C++ Smart Home Automation System

A console-based smart home simulation written in C++ that allows users to control multiple smart devices from a single menu.  
This project models real-world smart home components using clean object-oriented design.

---

## Features
- Centralized menu-driven smart home control system  
- Smart Light with ON/OFF state and adjustable brightness  
- Smart Thermostat with temperature control and power state  
- Smart Door Lock with lock/unlock functionality  
- Real-time status display for all devices  
- Input validation for safe and reliable interaction  

---

## Concepts Used
- Object-Oriented Programming (OOP)  
- Classes & Encapsulation  
- Constructors & Default Initialization  
- Input validation and control flow  
- Nested menus and looping structures  

---

## How to Run
1. Save the program as `main.cpp`  
2. Compile using a C++ compiler:  
   ```bash
   g++ main.cpp -o smart_home
