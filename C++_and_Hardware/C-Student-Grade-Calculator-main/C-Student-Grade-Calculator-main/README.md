# C++ Student Grade Calculator

A beginner-friendly C++ program to determine a student's grade based on their marks.  
This project demonstrates conditional logic, range-based grading, and user-friendly input/output handling.

---

## Features
- Accepts student marks as input  
- Calculates grade based on predefined ranges  
- Displays clear output with marks and corresponding grade  
- Handles invalid or out-of-range input with a default grade  

---

## Concepts Used
- Conditional statements (`if-else`)  
- Range checking for grading  
- Basic Input/Output (`cin`, `cout`)  
- Decision-making logic  

---

## How to Run
1. Save the program as `main.cpp`  
2. Compile using:  
   ```bash
   g++ main.cpp -o grade_calculator
