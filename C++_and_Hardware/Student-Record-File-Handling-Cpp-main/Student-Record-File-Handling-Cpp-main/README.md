# Student Record Management using File Handling (C++)

This project is a C++ console program that manages student records using **structures**, **file handling**, and **dynamic memory allocation**. It reads student data from text files, stores it, displays it in a formatted output, and then saves it back to separate output files.

---

## Features
- Reads student details from a file
- Supports RIS and EE department student records
- Stores:
  - Name  
  - Enrollment Number  
  - Semesters Passed  
  - CGPA  
  - Registered Courses
- Displays records in a clean console format
- Saves processed data back to output files
- Uses dynamic memory and structured data storage

---

## Concepts Used
- Structures in C++
- File Input / Output (ifstream & ofstream)
- Dynamic Memory Allocation (`new` and `delete`)
- Arrays & String Handling
