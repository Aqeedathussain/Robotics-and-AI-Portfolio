# PakistaniDiabetesPredictor

## Overview
This repository contains a machine learning model that predicts whether a person has diabetes (Yes/No) and also provides the probability percentage of having diabetes. The model is trained on the **Pakistani Diabetes Dataset**, which includes health indicators such as age, BMI, glucose level, blood pressure, pregnancies, insulin, skin thickness, and diabetes pedigree function. The project demonstrates binary classification with probability estimation using Python and Scikit-learn.

## Features
- Binary classification: Predicts "Diabetic" or "Non-Diabetic"
- Probability output: Shows the exact percentage chance of having diabetes
- Data preprocessing, feature scaling, and model evaluation
- Multiple model support (Logistic Regression, Random Forest, XGBoost)
- Performance metrics: Accuracy, Precision, Recall, F1-Score, ROC-AUC
